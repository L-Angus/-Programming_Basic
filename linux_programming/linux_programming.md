## 文件与IO
输入输出是主存与外被设备之间拷贝数据的过程
设备--》内存 （输入操作）
内存--》设备 （输出操作）

**高级IO** ANSI C库提供的标准IO库称为高级I/O，通常也称为带缓冲的IO；
**低级IO** 系统调用提供的IO操作接口,通常也称为不带缓冲的IO；

一个进程启动时，操作系统会打开三个文件描述符：标准输入、标准输出、标准错误。

fileno: 将文件指针转换为文件描述符；
fdopen: 将文件描述符转为文件指针。
